# Control de Asistencia GPS

## Desplegar en Vercel

1. Abre terminal en esta carpeta
2. Ejecuta: `vercel`
3. Sigue las instrucciones
4. ¡Listo!

## O desde la web

1. Ve a vercel.com/new
2. Arrastra esta carpeta
3. Deploy

---

**IMPORTANTE:** Si te pide configurar algo, usa estos valores:
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
